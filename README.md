# Your Digital Showcase

This repository contains a Vite + React + TypeScript starter with shadcn-ui and Tailwind CSS.

## Getting started

Prerequisites: Node.js and npm (or pnpm/yarn).

```sh
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm install
npm run dev
```

## Technologies

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn-ui

## Deployment

Deploy to Vercel, Netlify, or any static hosting that supports Vite-built sites. Configure environment variables (prefixed with `VITE_`) in your host if you use services like EmailJS.
